# Java Lang Ext
is a small library that contains utility functions to help the programmer
do more with less code.

* LINQ style operations
   * from(coll).map(..).group(..).join(..)
   * is lazy
* Helper methods
   * coalesce
   * humanize
   * str
   * isNullOrEmpty
   * tryParseNumber
